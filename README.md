# 🚀 Todo-list — Умный менеджер задач

**Внимание: Этот проект повышает личную эффективность на 25%!\**
*(\*По субъективным ощущениям автора. Ваши результаты могут отличаться, но попробовать стоит!)*

Этот пет-проект представляет собой простой, но мощный веб-интерфейс для управления вашими задачами. Серверная часть написана на быстром и надежном Golang, интерфейс — на чистом HTML/CSS, а логика фронтенда — на ванильном JavaScript.

---

## ✨ Возможности

*   **Создание и удаление задач:** Легко добавляйте и вычеркивайте выполненные дела.
*   **Минималистичный интерфейс:** Ничего лишнего, чтобы не отвлекать от работы.
*   **Локальный запуск:** Все ваши данные остаются на вашем компьютере.
*   **Невероятная скорость:** Бэкенд на Go обрабатывает запросы молниеносно.

---

## 🛠 Технологии

*   **Backend:** Golang (чистая стандартная библиотека `net/http`)
*   **Frontend:** Ванильные HTML5, CSS3, JavaScript (ES6+)
*   **Архитектура:** Классическая серверная рендеринг (SSR)

---

## ⚡ Быстрый старт

Запустить проект на своей машине проще простого. Вам нужен только установленный Go.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/your-username/your-todo-repo.git
    cd your-todo-repo
    ```

2.  **Запустите сервер:**
    ```bash
    go run main.go
    ```
    Сервер запустится на порту по умолчанию (вероятно, `:8080`).

3.  **Откройте браузер:**
    Перейдите по адресу [http://localhost:8080](http://localhost:8080) и начните продуктивную жизнь!
